após criar a pasta com, iniciar o projeto NodeJS:
npm init --yes

Instalar módulo do MySQL
nmp install mysql2 

Acrescentar o "tipo modulo" em package.json (abaixo do main)
"type": "module",

instalar modulo express
npm install express

Instalar módulo nodemon para reiniciar o servidor
npm install nodemon 

Adicionar em package.json -> "scripts": {
     "dev": "nodemon servidor/servidor.js"
  }

  Agora para rodar servidor usamos:
  npm run dev
